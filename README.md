# Be Happy — Catalog of Discourses by Swami Anubhavananda

Password-protected catalog of 5002 Vedanta talks across 956 playlists.

## Sections
- Upanishads — 28 playlists
- Bhagvad Gita — 24 playlists
- Prakarana Granthas — 52 playlists
- Meditation — 118 playlists
- Satsangs — 734 playlists

## How to use
1. Open `index.html` in a browser
2. Enter password to access
3. Browse talks by section → playlist → talk
